# Java real time chat application using web sockets

### App Breakdown:
- Backend
  - SpringBoot
  - SpringWebSocket
  - STOMP protocol
  - Thymeleaf

- Frontend
  - Thymeleaf
  - StockJS
  - stomp.js
  - JS
  - HTML/CSS
  - Bootstrap

## once app is runing you can view it on [app link once live](http://localhost:8080/chat)

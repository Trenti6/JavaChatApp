# Java real time chat application using web sockets

### App Breakdown:
- Backend
  - SpringBoot
  - SpringWebSocket
  - STOMP protocol
  - Thymeleaf

- Frontend
  - Thymeleaf
  - StockJS
  - stomp.js
  - JS
  - HTML/CSS
  - Bootstrap

## Once app is running you can view it -> ***[Here](http://localhost:8080/chat)***
